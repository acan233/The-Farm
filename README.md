# The Farm

一个使用 Godot 制作的 2D 像素风农场游戏 Demo。玩家可以在农场中移动、开荒、种植作物、浇水、收获资源，并与向导、动物和场景物件进行交互。

## 项目信息

- 引擎：Godot 4.7 Mono
- 项目名：farm4
- 主场景：`scenes/main_scene.tscn`
- 游戏类型：2D 农场模拟 Demo

## 已实现功能

- 玩家移动与方向动画
- 工具系统：斧头、锄头、浇水工具、作物种子
- 开荒耕地、移除耕地
- 玉米、番茄种植与成长
- 作物浇水后成长
- 树木、石头等可破坏物交互
- 原木、石头、作物、鸡蛋、牛奶等收集物
- 背包数量显示
- 宝箱提交收获物与奖励逻辑
- 小鸡、奶牛 NPC 行走与互动
- 向导对话与领取工具/种子
- 领取工具后的随机闲聊对话
- 昼夜时间系统和界面显示
- 背景音乐与音效
- 游戏菜单、开始游戏、保存与读取
- Windows Release 导出配置

## 操作说明

- `W A S D`：移动
- 鼠标左键：使用当前工具
- 鼠标右键：取消当前工具选择
- `E`：与可交互对象对话或互动
- `P`：保存游戏
- `Tab`：打开或关闭游戏菜单
- `Ctrl + 鼠标左键`：使用锄头移除已有耕地

## 运行项目

1. 安装 Godot 4.7 Mono。
2. 使用 Godot 打开本项目根目录。
3. 运行主场景或直接运行项目。

## 导出版本

当前项目已配置 Windows Desktop Release 导出预设。推荐导出为：

```text
farm4.exe
farm4.pck
```

这两个文件需要放在同一个文件夹中。发给别人测试时，可以把它们一起压缩成 zip。

## 项目结构

```text
addons/      插件目录
assets/      美术、字体、音频等资源
audio/       音乐和音效场景
dialogue/    对话气球和对话资源
resources/   存档、场景数据等资源脚本
scenes/      游戏角色、组件、关卡、UI 和物件场景
scripts/     全局管理器、输入处理、状态机等脚本
Tilesets/    TileSet 资源
```

## 说明

这是一个学习与展示用途的农场游戏 Demo，主要用于验证 Godot 2D 农场玩法、UI、对话、存档和导出流程。
